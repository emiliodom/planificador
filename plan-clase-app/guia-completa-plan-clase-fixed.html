Configurar Tailwind CSS con Plugin de Vite

Paso 1: Configurar el plugin de Tailwind en Vite. Edita el archivo vite.config.js: 

import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'
import tailwindcss from '@tailwindcss/vite'

export default defineConfig({
  plugins: [
    react(),
    tailwindcss(),
  ],
})

Paso 2: Importar Tailwind CSS en tu archivo principal de estilos. Crea o edita src/style.css: 

@import "tailwindcss";

Paso 3: Asegúrate de que el archivo CSS esté importado en src/main.jsx: 

import React from 'react'
import ReactDOM from 'react-dom/client'
import App from './App.jsx'
import './style.css' // Importar estilos

ReactDOM.createRoot(document.getElementById('root')).render(
  <React.StrictMode>
    <App />
  </React.StrictMode>,
)

¡Nuevo método más simple! Este enfoque usando el plugin de Vite es más directo y no requiere configurar PostCSS o Autoprefixer manualmente.

Configuración Completa de NocoDB

NocoDB es una plataforma que convierte cualquier base de datos en una API REST automáticamente. Es perfecta para prototipos y aplicaciones educativas.

Paso 2.1: Crear cuenta en NocoDB

  1. Visitar el sitio web: Ve a https://app.nocodb.com
  2. Registro: Haz clic en "Sign Up" (Registrarse)
  3. Datos personales: Completa el formulario con tu email y contraseña
  4. Verificación: Revisa tu email y confirma tu cuenta
  5. Acceder: Inicia sesión en la plataforma

Consejo: NocoDB ofrece un plan gratuito que es perfecto para aprender y desarrollar proyectos pequeños.

Paso 2.2: Crear una Base (Workspace)

Una "Base" en NocoDB es como un proyecto que contiene todas tus tablas relacionadas.

  1. Crear Base: En el dashboard principal, busca el botón + Create Base
  2. Nombre del proyecto: Escribe "Sistema Planes de Clase"
  3. Tipo de base de datos: Selecciona "Create a new database" para empezar desde cero
  4. Motor de base de datos: Elige "SQLite" (es más simple para aprender)
  5. Confirmar: Haz clic en "Create Base"

Importante: Guarda el nombre exacto de tu base, lo necesitarás más adelante para la URL de la API.

Paso 2.3: Crear tabla "planes_clase"

Ahora crearemos la tabla que almacenará toda la información de nuestros planes de clase.

  1. Acceder a tu base: Haz clic en la base que acabas de crear
  2. Nueva tabla: Busca el botón + Add new table
  3. Nombre de tabla: Escribe exactamente planes_clase
  4. Crear: Confirma con "Create Table"

Campos a crear en tu tabla:

Campo Tipo Obligatorio Descripción
docente SingleLineText ✅ Nombre del profesor
curso SingleLineText ✅ Materia o asignatura
tema SingleLineText ✅ Tema de la clase
fecha Date ✅ Fecha de la clase
duracion SingleLineText ❌ Duración (ej: "45 min")
objetivos LongText ✅ Objetivos de aprendizaje
contenido LongText ✅ Contenido a enseñar
recursos LongText ❌ Materiales necesarios
evaluacion LongText ❌ Métodos de evaluación
Cómo agregar cada campo:
  1. Haz clic en el botón "+" al final de las columnas
  2. Escribe el nombre del campo exactamente como aparece en la tabla
  3. Selecciona el tipo de dato correspondiente
  4. Marca "Required" si es obligatorio (✅)
  5. Haz clic en "Save"
  6. Repite para todos los campos

Paso 2.4: Crear Token de API

El token de API te permitirá conectar tu aplicación React con NocoDB de forma segura.

  1. Ir a configuración: Haz clic en tu avatar (esquina superior derecha) → "Account Settings"
  2. Tokens: En el menú lateral, selecciona "API Tokens"
  3. Nuevo token: Haz clic en "+ Create Token"
  4. Nombre del token: Escribe "plan-clase-app"
  5. Crear: Haz clic en "Generate"
  6. Copiar token: ¡IMPORTANTE! Copia el token inmediatamente y guárdalo en un lugar seguro

¡Atención! El token solo se muestra una vez. Si lo pierdes, tendrás que crear uno nuevo. Nunca compartas tu token públicamente.

Paso 2.5: Obtener endpoints de API REST v3

NocoDB genera automáticamente una API REST completa. Vamos a obtener los endpoints.

  1. Ir a tu base: Regresa a tu base "Sistema Planes de Clase"
  2. Menú de base: Haz clic en el menú desplegable de la base (flecha hacia abajo)
  3. REST APIs: Selecciona "REST APIs"
  4. Swagger UI: Se abrirá la documentación interactiva de tu API
  5. Autorizar: Haz clic en "Authorize" y pega tu token
  6. Explorar: Ahora puedes ver todos los endpoints disponibles

Endpoints principales que utilizaremos:

GET /api/v3/data/{baseId}/{tableId}/records - Obtener todos los planes
POST /api/v3/data/{baseId}/{tableId}/records - Crear nuevo plan
PATCH /api/v3/data/{baseId}/{tableId}/records - Actualizar plan
DELETE /api/v3/data/{baseId}/{tableId}/records - Eliminar plan

Paso 2.6: Obtener URL y Token para React

Solo necesitas dos valores para conectar tu aplicación React con NocoDB:

🔗 URL de la API (API_BASE)

En la documentación de Swagger, copia la URL completa que aparece en los ejemplos:

https://app.nocodb.com/api/v3/data/[BASE_ID]/[TABLE_ID]

⚡ Ejemplo: https://app.nocodb.com/api/v3/data/pp2zrs2rk71l250/m3qe8h6r9053bse

🔑 Token de API (API_TOKEN)

Usa el token que creaste en el paso anterior:

zlbClCcVqqRclQ4BnokVIs1Z6IVGFeBm0kRZ4ECv

⚡ Este token se copia desde: Account Settings → Tokens → Copy

📝 Código para tu componente React:

// Al inicio de tu componente LessonPlannerCRUD.jsx
const API_BASE = 'https://app.nocodb.com/api/v3/data/TU_BASE_ID/TU_TABLE_ID';
const API_TOKEN = 'TU_TOKEN_AQUI';

const headers = {
  'xc-token': API_TOKEN,
  'Accept': 'application/json',
  'Content-Type': 'application/json'
};

¡Importante! Reemplaza TU_BASE_ID, TU_TABLE_ID y TU_TOKEN_AQUI con tus valores reales.

Archivo: src/components/PlanForm.jsx

import { useState } from 'react';
import { planService } from '../services/nocodb';

function PlanForm({ onSave, planToEdit = null }) {
  const [formData, setFormData] = useState({
    docente: planToEdit?.docente || '',
    curso: planToEdit?.curso || '',
    tema: planToEdit?.tema || '',
    fecha: planToEdit?.fecha || '',
    duracion: planToEdit?.duracion || '',
    objetivos: planToEdit?.objetivos || '',
    contenido: planToEdit?.contenido || '',
    recursos: planToEdit?.recursos || '',
    evaluacion: planToEdit?.evaluacion || ''
  });

  const [isSubmitting, setIsSubmitting] = useState(false);
  const [errors, setErrors] = useState({});

  // Validar formulario
  const validateForm = () => {
    const newErrors = {};
    
    if (!formData.docente.trim()) newErrors.docente = 'El nombre del docente es obligatorio';
    if (!formData.curso.trim()) newErrors.curso = 'El curso es obligatorio';
    if (!formData.tema.trim()) newErrors.tema = 'El tema es obligatorio';
    if (!formData.fecha) newErrors.fecha = 'La fecha es obligatoria';
    if (!formData.objetivos.trim()) newErrors.objetivos = 'Los objetivos son obligatorios';
    if (!formData.contenido.trim()) newErrors.contenido = 'El contenido es obligatorio';
    
    setErrors(newErrors);
    return Object.keys(newErrors).length === 0;
  };

  const handleSubmit = async (e) => {
    e.preventDefault();
    
    if (!validateForm()) {
      alert('Por favor, completa todos los campos obligatorios');
      return;
    }

    setIsSubmitting(true);
    
    try {
      if (planToEdit) {
        // Actualizar plan existente
        await planService.update(planToEdit.id, formData);
        alert('✅ Plan actualizado exitosamente');
      } else {
        // Crear nuevo plan
        await planService.create(formData);
        alert('✅ Plan creado exitosamente');
        
        // Limpiar formulario después de crear
        setFormData({
          docente: '',
          curso: '',
          tema: '',
          fecha: '',
          duracion: '',
          objetivos: '',
          contenido: '',
          recursos: '',
          evaluacion: ''
        });
      }
      
      if (onSave) onSave();
      
    } catch (error) {
      console.error('Error:', error);
      alert('❌ Error al guardar el plan. Revisa la consola para más detalles.');
    } finally {
      setIsSubmitting(false);
    }
  };

  const handleChange = (e) => {
    const { name, value } = e.target;
    setFormData(prev => ({
      ...prev,
      [name]: value
    }));
    
    // Limpiar error del campo cuando el usuario empiece a escribir
    if (errors[name]) {
      setErrors(prev => ({
        ...prev,
        [name]: ''
      }));
    }
  };

  const handleReset = () => {
    if (confirm('¿Estás seguro de que quieres limpiar el formulario?')) {
      setFormData({
        docente: '',
        curso: '',
        tema: '',
        fecha: '',
        duracion: '',
        objetivos: '',
        contenido: '',
        recursos: '',
        evaluacion: ''
      });
      setErrors({});
    }
  };

  return (
    <div className="bg-white p-6 rounded-lg shadow-lg">
      <h2 className="text-2xl font-bold mb-6 text-gray-800">
        <i className="fas fa-edit mr-3 text-blue-600"></i>
        {planToEdit ? 'Editar Plan de Clase' : 'Nuevo Plan de Clase'}
      </h2>
      
      <form onSubmit={handleSubmit} className="space-y-6">
        {/* Información básica */}
        <div className="bg-gray-50 p-4 rounded-lg">
          <h3 className="text-lg font-semibold mb-4 text-gray-700">Información Básica</h3>
          <div className="grid grid-cols-1 md:grid-cols-2 gap-4">
            <div>
              <label className="block text-sm font-medium mb-2 text-gray-700">
                Docente <span className="text-red-500">*</span>
              </label>
              <input
                type="text"
                name="docente"
                value={formData.docente}
                onChange={handleChange}
                className={`w-full p-3 border rounded-lg focus:ring-2 focus:ring-blue-500 focus:border-blue-500 ${
                  errors.docente ? 'border-red-500' : 'border-gray-300'
                }`}
                placeholder="Ej: Prof. Juan Pérez"
                required
              />
              {errors.docente && <p className="text-red-500 text-sm mt-1">{errors.docente}</p>}
            </div>
            
            <div>
              <label className="block text-sm font-medium mb-2 text-gray-700">
                Curso/Materia <span className="text-red-500">*</span>
              </label>
              <input
                type="text"
                name="curso"
                value={formData.curso}
                onChange={handleChange}
                className={`w-full p-3 border rounded-lg focus:ring-2 focus:ring-blue-500 focus:border-blue-500 ${
                  errors.curso ? 'border-red-500' : 'border-gray-300'
                }`}
                placeholder="Ej: Matemáticas 5° Básico"
                required
              />
              {errors.curso && <p className="text-red-500 text-sm mt-1">{errors.curso}</p>}
            </div>
            
            <div>
              <label className="block text-sm font-medium mb-2 text-gray-700">
                Tema de la Clase <span className="text-red-500">*</span>
              </label>
              <input
                type="text"
                name="tema"
                value={formData.tema}
                onChange={handleChange}
                className={`w-full p-3 border rounded-lg focus:ring-2 focus:ring-blue-500 focus:border-blue-500 ${
                  errors.tema ? 'border-red-500' : 'border-gray-300'
                }`}
                placeholder="Ej: Fracciones y números decimales"
                required
              />
              {errors.tema && <p className="text-red-500 text-sm mt-1">{errors.tema}</p>}
            </div>
            
            <div>
              <label className="block text-sm font-medium mb-2 text-gray-700">
                Fecha <span className="text-red-500">*</span>
              </label>
              <input
                type="date"
                name="fecha"
                value={formData.fecha}
                onChange={handleChange}
                className={`w-full p-3 border rounded-lg focus:ring-2 focus:ring-blue-500 focus:border-blue-500 ${
                  errors.fecha ? 'border-red-500' : 'border-gray-300'
                }`}
                required
              />
              {errors.fecha && <p className="text-red-500 text-sm mt-1">{errors.fecha}</p>}
            </div>
          </div>
          
          <div className="mt-4">
            <label className="block text-sm font-medium mb-2 text-gray-700">
              Duración
            </label>
            <input
              type="text"
              name="duracion"
              value={formData.duracion}
              onChange={handleChange}
              className="w-full md:w-1/3 p-3 border border-gray-300 rounded-lg focus:ring-2 focus:ring-blue-500 focus:border-blue-500"
              placeholder="Ej: 45 minutos"
            />
          </div>
        </div>

        {/* Contenido pedagógico */}
        <div className="bg-blue-50 p-4 rounded-lg">
          <h3 className="text-lg font-semibold mb-4 text-blue-800">Contenido Pedagógico</h3>
          
          <div className="mb-4">
            <label className="block text-sm font-medium mb-2 text-gray-700">
              Objetivos de Aprendizaje <span className="text-red-500">*</span>
            </label>
            <textarea
              name="objetivos"
              value={formData.objetivos}
              onChange={handleChange}
              rows="3"
              className={`w-full p-3 border rounded-lg focus:ring-2 focus:ring-blue-500 focus:border-blue-500 ${
                errors.objetivos ? 'border-red-500' : 'border-gray-300'
              }`}
              placeholder="Describe qué deben lograr los estudiantes al finalizar la clase..."
              required
            ></textarea>
            {errors.objetivos && <p className="text-red-500 text-sm mt-1">{errors.objetivos}</p>}
          </div>

          <div className="mb-4">
            <label className="block text-sm font-medium mb-2 text-gray-700">
              Contenido a Desarrollar <span className="text-red-500">*</span>
            </label>
            <textarea
              name="contenido"
              value={formData.contenido}
              onChange={handleChange}
              rows="4"
              className={`w-full p-3 border rounded-lg focus:ring-2 focus:ring-blue-500 focus:border-blue-500 ${
                errors.contenido ? 'border-red-500' : 'border-gray-300'
              }`}
              placeholder="Detalla el contenido que vas a enseñar, actividades, ejemplos..."
              required
            ></textarea>
            {errors.contenido && <p className="text-red-500 text-sm mt-1">{errors.contenido}</p>}
          </div>
        </div>

        {/* Recursos y evaluación */}
        <div className="bg-green-50 p-4 rounded-lg">
          <h3 className="text-lg font-semibold mb-4 text-green-800">Recursos y Evaluación</h3>
          
          <div className="mb-4">
            <label className="block text-sm font-medium mb-2 text-gray-700">
              Recursos Necesarios
            </label>
            <textarea
              name="recursos"
              value={formData.recursos}
              onChange={handleChange}
              rows="3"
              className="w-full p-3 border border-gray-300 rounded-lg focus:ring-2 focus:ring-blue-500 focus:border-blue-500"
              placeholder="Lista los materiales, tecnología, libros, etc. que necesitarás..."
            ></textarea>
          </div>

          <div className="mb-6">
            <label className="block text-sm font-medium mb-2 text-gray-700">
              Estrategias de Evaluación
            </label>
            <textarea
              name="evaluacion"
              value={formData.evaluacion}
              onChange={handleChange}
              rows="3"
              className="w-full p-3 border border-gray-300 rounded-lg focus:ring-2 focus:ring-blue-500 focus:border-blue-500"
              placeholder="Describe cómo vas a evaluar el aprendizaje de los estudiantes..."
            ></textarea>
          </div>
        </div>

        {/* Botones */}
        <div className="flex flex-col sm:flex-row gap-4 pt-4 border-t border-gray-200">
          <button
            type="submit"
            disabled={isSubmitting}
            className="flex-1 bg-blue-600 text-white py-3 px-6 rounded-lg hover:bg-blue-700 disabled:bg-gray-400 font-semibold transition-colors"
          >
            {isSubmitting ? (
              <>
                <i className="fas fa-spinner fa-spin mr-2"></i>
                {planToEdit ? 'Actualizando...' : 'Guardando...'}
              </>
            ) : (
              <>
                <i className="fas fa-save mr-2"></i>
                {planToEdit ? 'Actualizar Plan' : 'Guardar Plan'}
              </>
            )}
          </button>
          
          <button
            type="button"
            onClick={handleReset}
            className="flex-1 sm:flex-none bg-gray-500 text-white py-3 px-6 rounded-lg hover:bg-gray-600 font-semibold transition-colors"
          >
            <i className="fas fa-eraser mr-2"></i>
            Limpiar
          </button>
        </div>
      </form>
    </div>
  );
}

import html2canvas from 'html2canvas';
import jsPDF from 'jspdf';

function PDFExport({ planData }) {
  const exportToPDF = async () => {
    const element = document.getElementById('plan-content');
    
    try {
      const canvas = await html2canvas(element, {
        scale: 2,
        useCORS: true,
        allowTaint: true,
        backgroundColor: '#ffffff'
      });

      const imgData = canvas.toDataURL('image/png');
      const pdf = new jsPDF('p', 'mm', 'a4');
      
      const imgWidth = 210;
      const pageHeight = 297;
      const imgHeight = (canvas.height * imgWidth) / canvas.width;
      
      pdf.addImage(imgData, 'PNG', 0, 0, imgWidth, imgHeight);
      pdf.save(`plan-clase-${planData.tema}.pdf`);
    } catch (error) {
      console.error('Error al generar PDF:', error);
    }
  };

  return (
    <div>
      <div id="plan-content" className="bg-white p-8">
        <h1 className="text-2xl font-bold mb-4">Plan de Clase</h1>
        <div className="grid grid-cols-2 gap-4 mb-4">
          <p><strong>Docente:</strong> {planData.docente}</p>
          <p><strong>Curso:</strong> {planData.curso}</p>
          <p><strong>Tema:</strong> {planData.tema}</p>
          <p><strong>Fecha:</strong> {planData.fecha}</p>
        </div>
        
        <div className="mb-4">
          <h2 className="text-lg font-bold mb-2">Objetivos</h2>
          <p>{planData.objetivos}</p>
        </div>
        
        <div className="mb-4">
          <h2 className="text-lg font-bold mb-2">Contenido</h2>
          <p>{planData.contenido}</p>
        </div>
        
        <div>
          <h2 className="text-lg font-bold mb-2">Recursos</h2>
          <p>{planData.recursos}</p>
        </div>
      </div>
      
      <button
        onClick={exportToPDF}
        className="mt-4 bg-red-600 text-white px-6 py-3 rounded-lg hover:bg-red-700"
      >
        <i className="fas fa-download mr-2"></i>
        Exportar a PDF
      </button>
    </div>
  );
}

export default PDFExport;
